## DefectDojo CLI
[![License](https://img.shields.io/badge/license-MIT-_red.svg)](https://opensource.org/licenses/MIT)

A CLI wrapper for [DefectDojo](https://github.com/DefectDojo/django-DefectDojo)

## Installation

Simply run:
```
$ python3 -m pip install defectdojo_cli
```

## Usage

```
$ defectdojo --help
```
